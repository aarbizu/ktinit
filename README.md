# KtInit

CLI to bootstrap kt projects.

### build and install
```
brew install gradle 
make install        ## installs at /usr/local/bin/ktinit
```

### run 
```
ktinit --help
mkdir ~/kotlin && cd ~/kotlin
ktinit -c --group-id demo --artifact-id kotlindemo
```
